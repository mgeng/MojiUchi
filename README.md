# Gina — 漫画オーサリングツール

ブラウザだけで動く、軽量な漫画オーサリングツールです。
コマ割り・素材画像の配置・吹き出し・テキスト入力までを 1 ページで完結させ、
プロジェクトファイル (`.mj`) として保存したり、PNG として書き出したりできます。

ビルドや外部依存（npm など）は不要で、`index.html` を開けばそのまま動きます。

## デモ

公開版を以下で利用できます。インストール不要、ブラウザだけで動作します。

<https://mgeng.github.io/gina/>

## 主な機能

- **マルチページ**: 最大 16 ページまでのページ送り／削除に対応
- **コマ割りテンプレート**: 1 コマ / 縦 2 分割 / 横 2 分割 / 4 コマ縦 / 6 コマ / 8 コマ
- **コマ編集**: 選択コマの上下・左右分割、コマ削除（隣接コマが空きを埋める）、輪郭線の太さ・コマ間の調整
- **素材画像 (コマ内画像)**: コマをダブルクリックで画像を配置。ドラッグで移動、ホイールで拡縮、Shift+ホイールで回転
- **オーバーレイ画像**: ウィンドウへ画像をドラッグ&ドロップすると、コマをまたいで上から重ねる画像として配置
- **吹き出し**: 16 種類の形状（横書き用 / 縦書き用）から選択、リサイズ、左右・上下反転に対応
- **テキスト / モノローグ**: 横書き・縦書き、フォント・サイズ・行間を切り替え可能
- **フォント同梱**: 漫画用に定番の **源暎アンチック Pv6 / Nv6**、および **851チカラヅヨク**
- **プロジェクト保存**: `.mj` 形式（中身は ZIP）で全ページのレイアウト・テキスト・画像をまとめて保存／読み込み
- **PNG 書き出し**: ページ単位で PNG 出力
- **メモ**: セリフ案などをページごとに置いておけるサイドメモ

## 使い方

1. ヘッダーの **開く** から `.mj` プロジェクトを読み込むか、空のページから開始します。
2. 右側インスペクターの **テンプレート** からコマ割りを選びます。コマをクリックして選択 →
   **上下に分割 / 左右に分割 / コマを削除** で形を整えられます。
3. コマを **ダブルクリック** すると、素材画像を選んで配置できます。配置後はドラッグ移動 /
   ホイール拡縮 / Shift+ホイールで回転。
4. ページ上で **右クリック** → メニューから以下を追加できます。
   - **テキストを追加** … 通常のテキストレイヤー
   - **モノローグ** … モノローグ用テキスト
   - **フキダシ** … 形状を選んで配置（リサイズ・反転可）
   - **画像を上から追加** … コマをまたいで重ねるオーバーレイ画像
5. ウィンドウへ画像ファイルを **ドラッグ&ドロップ** すると、その位置を中心にオーバーレイ画像として追加されます。
6. ヘッダーの **保存** で `.mj` を上書き保存、**PNG書き出し** で現在のページを PNG 出力できます。

## ショートカット

| キー | 動作 |
| --- | --- |
| `Ctrl` + `O` | 開く |
| `Ctrl` + `S` | 上書き保存 |
| `Ctrl` + `Shift` + `S` | 名前を付けて保存 |
| `Ctrl` + `E` | PNG 書き出し |
| `Ctrl` + `←` / `→` | ページ移動 |
| `←` `→` `↑` `↓` | 選択中レイヤーを 1px 移動 |
| `Ctrl` + `↑` / `↓` | 選択中テキストの文字サイズ変更 |
| `Delete` | 選択中のコマ／レイヤーを削除 |

## ファイル形式 (`.mj`)

`.mj` は実体が ZIP のプロジェクトファイルで、以下の構成です。

```
pages/<N>/text.json       # テキスト・モノローグ・吹き出しレイヤー
pages/<N>/panels.json     # コマ割りと素材画像のレイアウト
pages/<N>/memo.txt        # メモ
pages/<N>/materials/      # コマ素材画像
pages/<N>/overlays/       # 上から重ねるオーバーレイ画像
```

すべての画像・テキストはこのファイル内に同梱されます。サーバーへは何もアップロードされません。

## ローカルで動かす

純粋な静的ファイルなので、リポジトリを取得して `index.html` を開くだけで動作します。

```bash
git clone https://github.com/mgeng/gina.git
```

`file://` でも基本的に動きますが、ブラウザによってはローカルフォントの読み込みが制限される
場合があります。その場合は任意の静的サーバー（VS Code の Live Server 拡張など）で開いてください。

## ディレクトリ構成

```
.
├── index.html              # エントリポイント
├── src/
│   └── main.js             # アプリ本体（バニラ JS）
├── styles/
│   └── style.css           # スタイル
├── assets/
│   ├── fonts/              # 同梱フォント
│   ├── bubbles/            # 吹き出し画像（horizontal / vertical）
│   └── vendor/
│       └── jszip.min.js    # .mj (ZIP) の読み書き
├── scripts/                # 吹き出し素材を生成する補助スクリプト
├── LICENSE
└── NOTICE.md
```

## 動作要件

- 最近の Chrome / Edge / Firefox / Safari
- 画像処理・保存はすべてブラウザ内で完結します。外部サーバーへの送信はありません。
