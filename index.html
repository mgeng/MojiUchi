メモ
ここに画像 / プロジェクト(.mj) / テキスト(.json) / メモ(.txt)をドラッグ&ドロップ、または「開く」から選択

ページレイアウト

コマをクリックで選択 → 分割ボタン。

素材の操作

ドラッグで移動 / ホイールで拡縮 / Shift+ホイールで回転

操作

画像上で右クリック → 「テキストを追加」で文字を追加。左クリックは選択のみ(何もない場所をクリックすると画像が選択され、Delete で画像のみ削除(テキストは保持))。

ショートカット: Ctrl+O 開く / Ctrl+S 保存 / Ctrl+E PNG書き出し / Ctrl+←→ ページ移動

選択中: 矢印キーで1px移動 / Ctrl+↑↓ で文字サイズ変更

選択中のテキスト

ドラッグで位置移動、ダブルクリックで本文を直接編集。
